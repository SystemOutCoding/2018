## 개요
올해도 Build 2018 컨퍼런스가 열렸습니다. 인공지능 이야기가 주로 나왔지만 IoT와 각종 클라우드 기술과 개발 툴 관련 기술 등 많은 내용들이 발표 되었습니다. 이 세션에서는 ‘모든 개발자들은 이제부터 AI개발자’라는 선언을 한 Microsoft Build Conference 2018의 전체 내용을 요약해 드립니다.
